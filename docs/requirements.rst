Requirements
============

Django
------
This package requires Django 3.2 or newer. The maintained support range is Django 3.2 through 6.0.

Python
------
This package requires Python 3.8 or newer. The maintained support range is Python 3.8 through 3.13, subject to what the selected Django version supports. See also `What Python version can I use with Django?`_.
